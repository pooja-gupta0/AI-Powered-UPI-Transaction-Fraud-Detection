# AI-Powered-UPI-Transaction-Fraud-Detection
Developed a Machine Learning-based UPI Fraud Detection System to identify fraudulent digital payment transactions and improve transaction security. The project involved collecting and preprocessing transaction data, handling missing values, performing feature engineering, and analyzing transaction patterns to distinguish legitimate and fraudulent activities. Implemented the Random Forest classification algorithm to build a predictive model capable of detecting suspicious transactions with high accuracy. The model was evaluated using performance metrics such as accuracy, precision, recall, F1-score, and confusion matrix to ensure reliable fraud detection. This project strengthened my skills in data preprocessing, exploratory data analysis (EDA), feature engineering, supervised machine learning, model evaluation, and fraud analytics using Python, Pandas, NumPy, Scikit-learn, and Matplotlib.


Technologies Used:
Python | Pandas | NumPy | Scikit-learn | Random Forest | Matplotlib | Machine Learning | Data Analysis
